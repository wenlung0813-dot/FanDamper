# Fire Damper Thermal Flow Model Documentation

## Architecture Overview

This repository contains a comprehensive thermal flow model for fire dampers. The main components include:

- **Thermal Flow Model**: A Python simulation of thermal behavior in fire dampers.
- **Design Documentation**: Detailed principles of the thermal flow model and its design decisions.
- **Simulation Scripts**: Python code organized for easy execution and testing.

## Getting Started

To run the model, ensure you have Python installed along with necessary dependencies. You can start simulations by executing the provided scripts in the `simulation` directory.

## Structure

The repository contains the following directories:

- `/docs`: Documentation files including design documents.
- `/simulation`: Code files for simulation.

## Usage

1. Clone the repository: `git clone <repo-url>`
2. Navigate to the simulation directory.
3. Run the simulation scripts.
